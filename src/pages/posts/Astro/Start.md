# Hello, Astro!  

这是我的第一篇文章，欢迎来到 Astro 的世界！

## 搭建 Astro 环境

> 1. 安装 Node.js `v18.17.1` 或 `20.3.0` 或更高版本。不支持 ～～v19～～  
> 2. 编辑器：推荐使用 VS Code 并安装 Astro 的官方扩展
> 3. 安装 Astro CLI `npm install -g astro`
